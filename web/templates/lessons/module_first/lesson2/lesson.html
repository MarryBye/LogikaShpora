Форматування тексту

За допомогою конкатенації (з'єднання)

    
        a = "Hello"
        b = "World"
        print(a + b)

Потрібно бути обережним із типами даних: все має бути рядком, якщо ви хочете вивести саме рядок. Або все має бути числами, якщо ви виводите, наприклад, суму чисел

    
        a = "Hello" # string - рядок
        b = 5       # integer - число
        print(a + str(b))

За допомогою методу .format()

    
        a = "Hello {0}!".format("world")
        print(a)

За допомогою методу f-string

    
        a = "world"
        print(f"Hello {a}!")

За допомогою шаблонів

    
        a = "world"
        b = "Hello %s!" % a # на місце %s буде підставлена змінна a
        print(b)

Декілька шаблонів

    
        name = "Oleg"
        meters = 5
        time = 3
        text = "%s, ви пройшли %s метрів за %s секунд!" % (name, meters, time)
        print(text)
