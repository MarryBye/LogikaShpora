Типи даних

У Python як в будь якій мові програмування існують типи даних. Простими словами, це те, як представлені наші дані, що з ними можна робити. Програміст може створювати власні типи даних, а є стандартні, такі як:

  • Integer (int) - ціле число
  • String (str) - рядок
  • Floating-point number (float) - число із комою
  • Boolean (bool) - логічне значення (True/False)
  • List - список
  • Dictionary (dict) - словник
  • Set - множина
  • Tuple - кортеж

У кожного типа даних є свої можливості та вони всі необхідні під час створення програм. Але зараз розглянемо перші три типа даних.

Вивести суму чисел:

    
        print(5 + 0.5)      # int + float
        print(1 + 1)        # int + int
        print(3.2 + 0.48)   # float + float

Між числами все просто - робити можна усі математичні дії та вивести їх легко. Але коли ми починаємо робити дії між повністю різними типами даних - потрібно бути обережним. Наприклад, ми не можемо знайти суму між рядок та числом.

Приклади помилок:

    
        print("5" + 0.5)            # ERROR, перше число - рядок, томущо у лапках
        print(1 + "Hello world!")   # ERROR

Але коли типи даних однакові, частіше за все операція буде виконана успішно, хоча не завжди (залежить від типа даних)

Інші приклади:

    
        print("5" + "3")            # виведе "53"
        print("Hello" + "world!")   # виведе "Hello world!"

В Python можна легко перетворити будь які дані в інший тип за допомогою таких функцій, як:

  • int() - перетворити в число
  • str() - перетворити в рядок
  • float() - перетворити у число із комою

Приклади перетворення:

    
        print(int("5") + int("3"))              # виведе 8
        print(int("3") + 1)                     # виведе 4
        print(int("Hello") + int("world!"))     # ERROR, не всі рядки можна перетворити в число, тільки якщо те, що там знаходиться підходить
        print(str(5) + str(3))                  # виведе "53"
        print(float("5.3") + float("1.1"))      # виведе 6.4
        print(float("5,3") + 1.1)               # ERROR, не можна перетворити перше число у float - має стояти крапка, а не кома.
